Remove some unnecessary headers from the pages.

== Description ==

Remove some unnecessary headers from the pages.

These are:

* link rel="EditURI" type="application/rsd xml" title="RSD"
* link rel="wlwmanifest" type="application/wlwmanifest+xml"
* link rel="shortlink"
* meta name="generator" content="WordPress 3.4.2"
* RSS feed stuff
